Pipelines

  • Processing items of the same type
  • Small tools/services
  • Fixed sequence of operations
  • Well established communication interfaces
  • Buffers

Advantages

  • No cyclic dependencies
  • Parallelism
  • Laziness
  • Immutability
  • Debugging

Characteristics

UNIX pipe

  • Processing item: lines
  • Small tools
  • Communication interface: text
  • Buffer
  • Lazy and parallel
  • Immutable
